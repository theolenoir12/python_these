VALIDATION DES PREMIERES VIES ET DES LOIS DE DEGRADATION - V10
================================================================
Mise a jour : 15 juillet 2026
Strategie de reference : RB2, setpoints FC=0,59 et ELY=0,49
Pas de temps : 1 h ; cout unifie = degradation + 3 EUR/kWh * EENS

1. STATUT DE LA VALIDATION
--------------------------
La validation distingue trois niveaux :
- verification : equations, unites, comptages et accumulation numerique ;
- ancrage : un coefficient est relie a une mesure ou cible publiee ;
- validation externe : la premiere vie est comparee a un essai suffisamment
  proche en technologie, temperature, densite de courant et profil d'usage.

Une cible DOE n'est pas une identification experimentale. Une duree de vie
extrapolee hors du domaine d'essai n'est pas declaree validee, meme si son
ordre de grandeur est plausible.

2. CORRECTION BATTERIE IDENTIFIEE
---------------------------------
La LUT Common/Cumulative_degradation_bat.txt vient de Maheshwari et al. :
- Sony US18650V3, chimie NMC, capacite de reference 2,15 Ah ;
- temperature regulee a 20 degC ;
- vieillissement par cyclage uniquement ;
- LUT delta_1C(SoC) et points de courant disponibles a 1C et 2C.

La constante 2,15 du code est donc la capacite de la cellule de reference,
et non une constante empirique sans unite.

L'article definit le facteur de courant :
  psi(i) = i                         pour 0 <= i <= 1C
  psi(i) = 1 + 0,2956 (i - 1)       pour 1C < i <= 2C
avec psi(0)=0, psi(1C)=1 et psi(2C)=1,2956.

Le code V10 utilisait a tort psi=1 sur tout [0,1C]. Comme le micro-reseau
fonctionne surtout vers 0,04C, l'usure batterie etait surestimee d'un facteur
proche de 1/i sur cette contribution. La loi a ete corrigee et testee aux
points 0C, 0,5C, 1C et 2C.

Cette correction est la seule modification de calibration retenue. Elle est
imposee par la source primaire et par l'equation deja ecrite dans le manuscrit ;
elle n'a pas ete choisie pour ameliorer RB2 ou RB2(SoH).

3. PREMIERE VIE BATTERIE
------------------------
Sur l'horizon Pareto de 25 ans :
- fin de vie SoH=0,70 non atteinte ;
- 3 149,7 EFC par energie dechargee ;
- 3 320,0 EFC par throughput ;
- C-rate moyen tous pas : 0,0303C ; p95 : 0,1210C ;
- SoC moyen : 0,7618 ; plage : 0,2000 a 0,9950.

Une simulation prolongee a 50 ans donne la premiere fin de vie de cyclage :
- duree calendaire : 345 459 h = 39,436 ans ;
- 4 949,6 EFC par energie dechargee ;
- 5 217,2 EFC par throughput ;
- C-rate actif moyen : 0,03936C ; p95 actif : 0,13316C ;
- SoC moyen : 0,7477 ;
- SoH avant reset : 0,700004.

Verdict batterie : la loi et son changement d'echelle sont verifies et traces
a la source. La premiere vie de 39,4 ans est une VIE DE CYCLAGE MODELISEE, pas
une duree de vie physique validee. Deux extrapolations dominent :
- psi est interpole lineairement de 0 a 1C alors que les donnees de calibration
  sont a 1C et 2C ; le fonctionnement reel est vers 0,04C ;
- le vieillissement calendaire est nul, choix admissible pour l'optimisation
  court terme de Maheshwari, mais limitant sur 25 a 40 ans.

Le seuil SoH batterie reste volontairement fixe a 0,70. Aucun terme calendaire
arbitraire n'est active dans le nominal. Une loi calendaire ne devra etre
ajoutee qu'avec une temperature et une dependance au SoC explicitement
calibrees ; sa sensibilite devra etre presentee separement.

4. PREMIERE VIE PEMFC
---------------------
Resultats RB2 :
- duree calendaire : 50 123 h = 5,722 ans ;
- temps ON : 27 263 h ; EFPH : 20 295,5 h ;
- energie fournie : 31 325,8 kWh ;
- 2 192 demarrages factures ; run moyen : 12,44 h ;
- j moyen ON : 0,565 A/cm2 ; j95 : 0,581 A/cm2 ;
- perte totale : 86,023 mV = 3,155 microV/h ON ;
- perte irreversible : 1,535 microV/h ON ;
- 51,0 % de la perte finale vient des demarrages.

La cible DOE backup power 1-10 kWe est 10 000 h. Liu et al. rapportent
26 300 h continues a 0,8 A/cm2, 110 mV perdus et environ 4,2 microV/h.
McCay et al. mesurent 1,2 microV/h irreversible en statique et 4,8 microV/h
sous profil dynamique, sur 1 500 h ; la part reversible domine. Des essais
start-stop LT-PEM rapportent typiquement environ 15 a 65 microV/cycle selon le
protocole, ce qui place le scenario V10 de 20 microV/demarrage dans la plage,
sans en faire une calibration specifique du stack GENIAL.

Verdict PEMFC : premiere vie et taux total plausibles, proches de l'essai long
continu de Liu. La decomposition reversible/irreversible est ancree, mais
l'extrapolation de McCay sur 1 500 h et la penalite de demarrage restent des
incertitudes a balayer. Aucun recalage nominal supplementaire n'est justifie
par une source suffisamment proche.

5. PREMIERE VIE PEMWE
---------------------
Resultats RB2 :
- duree calendaire : 134 023 h = 15,299 ans ;
- temps ON : 32 235 h ; EFPH : 16 220,4 h ;
- energie absorbee : 256 271,4 kWh ;
- 5 528 demarrages ; run moyen : 5,83 h ;
- j moyen ON : 1,855 A/cm2 ; j95 : 1,939 A/cm2 ;
- aucune heure au-dessus de 2 A/cm2 ;
- perte totale : 205,102 mV = 6,363 microV/h ON ;
- irreversible : 4,192 microV/h ON.

Decomposition finale :
- irreversible : 65,9 % ; demarrages : 31,5 % ;
- maintien : 2,0 % ; reversible restant : 0,5 %.

La cible/statut DOE 2022 donne 4,8 mV/1 000 h et environ 40 000 h jusqu'a
+10 % de tension. Le modele atteint 32 235 h ON : le socle irreversible est
proche de DOE, et l'ecart vient principalement du terme de demarrage.
Rakousky et al. ancrent la recuperation reversible et la sensibilite aux
profils jusqu'a 2 A/cm2. L'acceleration au-dessus de 2 A/cm2 est inactive pour
la premiere vie RB2 et ne contribue donc pas a ce resultat.

Verdict PEMWE : ordre de grandeur acceptable mais validation partielle. Le
coefficient 11,7 microV/demarrage est identifie avec la structure transitoire
sur les modes Rakousky, pas par un essai start-stop long independant. Il doit
rester un parametre de sensibilite ; le modifier pour atteindre exactement
40 000 h constituerait un ajustement circulaire.

7. RESULTAT RB2(SoH) APRES VALIDATION
-------------------------------------
Meilleur cout unifie teste :
- strength_FC=0,025 ; strength_ELY=0 ; shape=2 ;
- LPSP=0,66738 % ; degradation=24,63466 kEUR ;
- cout unifie=35,11909 kEUR, soit -0,79 % par rapport a RB2.

Compromis de degradation sous l'iso-cout RB2 :
- strength_FC=strength_ELY=0,05 ; shape=1 ;
- LPSP=0,69768 % ; degradation=24,39285 kEUR ;
- cout unifie=35,35326 kEUR ;
- degradation -1,23 %, LPSP +0,0164 point.

Conclusion scientifique : le SoH applique uniquement aux setpoints H2 produit
un effet reel et un petit front LPSP/degradation, mais le gain de cout unifie
reste inferieur a 1 %, tres loin du seuil de plusieurs pourcents. Dans ce cas,
l'augmentation de RB2 par le seul SoH est donc peu concluante. Ce resultat ne
doit pas etre masque par un recalage statique, puisque le parent RB2 a ete
re-optimise et reste (0,59 ; 0,49).
